# Introduce_Me
personal project

## Introduction
This is a self-introduction Android application.
It consists of three activities.
1. Sign In page
2. Sign Up page
3. Introduction page

## Development Environment
- Android Studio @Giraffe

## Application Version
- minSdk 24
- targetSdk 33
